"풀스택 서비스 프로그래밍 프로젝트"

1.	주제
CampusPool(카풀 매칭 커뮤니티 앱앱)

2.	추진 목적
본 프로젝트는 특정 시간대(1, 2교시, 점심시간 등)에 대중교통 이용 대기 시간이 길어 불편함을 겪는 경희대학교 국제캠퍼스 학생들의 이동 편의성을 개선하기 위해 실시간 카풀 매칭 시스템을 구현하고자 한다. 
본 시스템은 운전자와 탑승자를 실시간으로 매칭하여 대중교통 대기 시간을 단축하고, 이동 효율성을 높임으로써 학생들의 시간적 부담을 줄이며 학생들 간의 교류를 활성화하는 것을 목표로 한다. 또한, 사용자 간의 원활한 소통과 직관적인 매칭 과정을 통해 편리하고 신속한 이동 서비스를 제공하고자 한다.

3.	주요 기능
-	사용자 등록 및 로그인
-	게시물 등록 및 목록 조회
-	예약 및 예약확인인
-	실시간 채팅
-	프로필 관리

4.	창작/ Clone 여부: 창작

5.	클라이언트 타입: 앱 개발
